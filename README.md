### Requirements

- PHP: ^8.0
- Laravel: ^10

### Installation

1. Install CRUDify via Composer
    ```bash
    composer require lifehack/crudify
    ```

2. Publish CRUDify package resources.
    ```bash
    php artisan crud:start
    ```
